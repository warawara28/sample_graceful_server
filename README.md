# sample_graceful_server

This is a sample server that can stop gracefully.


## How to fmt

run this command.

```bash
$ make fmt
```

## How to build

run this command.

```bash
$ make build
```

## How to run

After build, run this command.

```bash
$ ./sample_graceful_server
```
